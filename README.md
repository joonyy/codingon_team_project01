# 홈페이지 개발 기획서

## 1. 소개
- 프로젝트 개요: 신발 쇼핑몰
- 목적 및 목표: 사용자가 봤을때 이쁜 쇼핑몰
- 대상 사용자: 20대 ~ 30대

## 2. 프로젝트 개발 일정
- 계획 수립 날짜: 3-11
- 개발 시작 날짜: 3-12 ~ 3-19
- 예상 완료 날짜: 3-20

## 3. 팀 정보
- 팀 이름: 남정내들
- 팀원
  - 김준영 : 리뷰, 프로젝트 관리
  - 배동명 : 프로젝트 관리
  - 이준규 : 일정관리, 리더

## 4. 기능 요구 사항
- 필수 기능 : 상품 디스플레이, 회원가입, 로그인(클릭으로 화면만 전환), 모션그래픽
- 선택적 기능 : 찜하기, AOS활용, 가로스크롤(좌우 투명하게), 상품 검색, 카테고리
- 확장 가능한 기능 : 신발 커스텀, 스크롤 방향

## 5. 기술적 요구 사항
- 개발 플랫폼 : vscode
- 프로그래밍 언어 : HTML, CSS, JS
